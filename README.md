# Aerolínea TAE ✈️

Una aplicación moderna de gestión de vuelos y reservas diseñada para Android, enfocada en ofrecer una experiencia de usuario fluida y segura.

## ✨ Características Principales

*   **🏠 Inicio Dinámico**: Visualización de destinos populares y ofertas especiales cargadas en tiempo real desde la nube.
*   **🔍 Búsqueda de Vuelos**: Buscador avanzado con soporte para trayectos de "Solo ida" o "Ida y vuelta", selección de pasajeros y validación de fechas.
*   **🎫 Gestión de Reservas**: Consulta de viajes programados y sistema de **Check-in online** mediante código PNR (Localizador).
*   **👤 Perfil de Usuario**: Gestión de datos personales y visualización del estado de cuenta.
*   **🛡️ Seguridad (MFA)**: Protección de cuenta mediante autenticación de dos factores configurable:
    *   Código PIN de seguridad.
    *   Autenticación biométrica (Huella dactilar).
